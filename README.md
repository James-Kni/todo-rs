### Dev commands
Gotta install them node dependencies :(
```sh
npm install
```

Build CSS live with 
```sh
npm run css:dev
```

Run with (Requires [cargo-watch](https://github.com/watchexec/cargo-watch))
```sh
cargo watch -x run
```

### Release commands
Build release binary
```sh
cargo build -r
```

Build minified CSS with
```sh
npm run css
```
